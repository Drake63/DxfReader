Диалоги

В этом разделе:

Диалог "Профили"
Используется для создания и редактирования пользовательских профилей, а так же используется для просмотра настроек текущего внутреннего профиля во время выполнения программы.
Диалог "Точки"
Используется для изменения или удаления объектов POINT, если они есть в документе
Диалог "О программе"
Описание диалога "О программе"
