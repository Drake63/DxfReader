Общая информация о профилях

Профиль применяется при открытии файла DXF. В нем указываются параметры, с которыми будет создаваться документ. Профиль, который будет применён, выбирается на панели инструментов или в меню Профиль.

В программе есть три основных профиля, а также возможность создавать пользовательские

Опции, которые можно настраивать в профиле:

Имя профиля
Эта опция обязательная, требуется для его идентификации, содержит название профиля.
Создавать LWPOLYLINE
При включении этой опции будут создаваться объекты LWPOLYLINE вместо POLYLINE. Они – являются более простыми объектами, т.к. не содержат объектов VERTEX для обозначения своих вершин в отличии от POLYLINE, а просто перечисляют их координаты последовательно. Поэтому запись таких объектов в файле будет короче, и соответственно сам файл будет иметь меньший размер.
Преобразовывать полилинии
При включении этой опции будут преобразовываться полилинии, вершины которых содержат коэффициент искривлённости, в более простые объекты - LINE, ARC и POLYLINE, вершины которых не содержат коэффициента искривлённости.
Пропускать все объекты
При включении этой опции, будут отображаться все типы линии, а не только основная, разметка и незамкнутый контур. Линии отличные от перечисленных будут создаваться как основные.
Все на начало координат
При включении этой опции все объекты будут перемещены так, чтобы нижний левый угол изображения соответствовал точке с координатами (0, 0), т.е. началу координат. Соответственно все объекты будут смещены на соответствующий вектор.
Преобразовывать эллипсы
При включении этой опции, эллипсы и эллиптические дуги будут преобразовываться в полилинии. Если она будет включена, то дополнительно потребуется указать точность, с которой будет производится преобразование.
Создать R13
При включении этой опции, документ будет сохраняться в как файл DXF версии R13 (AC1012), по умолчанию он сохраняется как файл версии R11 (AC1009). Это требуется если нужно редактировать файл в CAD программах, чтобы такие объекты как SPLINE или ELLIPSE распознавались ими корректно. При сохранении, в итоговый файл добавляются дополнительные данные, чтоб файл соответствовал версии R13.
