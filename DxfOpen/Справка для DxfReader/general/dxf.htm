Краткая информация о формате DXF

DXF – это файл обмена данными AutoCAD, разработанный компанией Autodesk для обмена данными между другими CAD системами. Впервые этот формат был выпущен в 1982 году. Файлы DXF используются практически любой системой CAD.

Формат DXF представляет собой текстовой файл, содержащий описание графического документа. Файл содержит в себе 7 секций, но графические примитивы находятся только в двух из них – BLOCKS и ENTITIES.
